# Code of Conduct

## Our Commitment

为了让 Digital Avatar 社区成为一个开放、专业、可协作的环境，所有参与者都应以尊重、务实和建设性的方式进行交流与协作。

## Expected Behavior

- 尊重不同背景、经验和观点
- 聚焦问题本身，避免人身攻击
- 以证据、案例和可执行建议推动讨论
- 对安全、隐私、合规相关问题保持审慎
- 对评审意见给出明确依据，而不是泛泛否定

## Unacceptable Behavior

- 骚扰、歧视、侮辱或威胁言行
- 恶意贬低贡献者或用户
- 泄露他人隐私、敏感信息或安全细节
- 明知存在风险仍推动不负责任的发布或变更
- 在讨论中反复引入与项目目标无关的对抗性内容

## Scope

本准则适用于仓库 Issues、Pull Requests、Discussions，以及其他与项目协作直接相关的公开交流场景。

## Enforcement

项目维护者有权对违反行为准则的内容进行删除、警告、限制参与或移除协作权限。

## Reporting

如需报告不当行为，可通过 [SECURITY.md](SECURITY.md) 中的联系方式联系维护者；若问题不涉及安全，也可通过 Issue 或其他维护渠道反馈。
